Data

Per-player data instance. Writable on the server, read-only on the client. Use Get() for merged reads and GetPersisted() for saved ProfileStore values.

Functions​

Get​

</>
Data:Get(
path: PathToken?--

Path tipado ou array legado. Omita para a raiz merged.

) → any

Reads the merged value (persisted + transient overlay). Omit path for the full root.

GetPersisted​

Query
</>
Data:GetPersisted(path: PathToken?) → any

Reads the saved ProfileStore value only (ignores transient overlay).

GetTransient​

Query
</>
Data:GetTransient(path: PathToken?) → any?

Reads the transient overlay at path, if present.

HasTransient​

Query
</>

Returns whether a transient overlay exists at or below the path.
