# Local Rojo test sandbox (template)

Copied to `Test/` by `scripts/setup-test-env.ps1`. The `Test/` folder is gitignored.

## Quick start

From repo root:

```powershell
.\scripts\setup-test-env.ps1
cd Test
rojo serve
```

Connect Roblox Studio via the Rojo plugin. `Packages.dataservicev2` maps to `../src`.
